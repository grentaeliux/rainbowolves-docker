Technical Information

PaperMC

Paper is a Minecraft game server based on Spigot, designed to greatly improve performance and offer more advanced features and API.

GitHub Page

Geyser

Geyser is a middleware, which translates all the incoming and outgoing packets. This enables Minecraft Bedrock Edition clients to connect to Minecraft: Java Edition servers.

GitHub Page

Floodgate

Floodgate is a hybrid mode plugin which allows for Minecraft: Bedrock Edition players to join Minecraft: Java Edition servers without needing a Minecraft: Java Edition account.

GitHub Page

eBackup

eBackup helps with creating backups of Minecraft server data, ensuring that crucial world data is not lost.

GitHub Page

Dynmap

Dynmap is a web-based map for Minecraft servers that updates in real time. It can be used to view and navigate the world map from a web browser.

GitHub Page

WorldEdit

WorldEdit is an in-game map editor for Minecraft, providing an extensive selection of powerful building and editing tools.

GitHub Page
